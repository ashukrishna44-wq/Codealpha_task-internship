# Task 2 — Phishing Awareness Training

## Overview
An interactive phishing awareness training module built as a single-page web application. Designed to educate users about phishing attacks, how to identify them, and how to stay protected online.

## Features
- What phishing is and why it's dangerous
- Common types of phishing attacks (Email, Spear, Smishing, Vishing, etc.)
- How to identify phishing attempts
- Real-world phishing examples
- Prevention tips and best practices
- Interactive quiz to test awareness

## How to Run
Simply open `phishing_awareness.html` in any web browser — no installation or server required.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose
This project was completed as **Task 2** of the **CodeAlpha Cybersecurity Internship** to demonstrate knowledge of social engineering threats and the ability to communicate security awareness effectively to end users.
