# CodeAlpha Cybersecurity Internship

This repository contains all tasks completed during my **Cybersecurity Internship at CodeAlpha**.  
Each task demonstrates practical cybersecurity skills including network analysis, social engineering awareness, and secure coding practices.

---

## Tasks Overview

### ✅ Task 1 — Network Intrusion Detection System
> Built a Python-based network packet sniffer and intrusion detection system.

- Captured and analyzed live network traffic
- Detected suspicious packets and potential intrusion attempts
- Logged findings to a file for review
- **Tools:** Python, Scapy, Socket

---

### ✅ Task 2 — Phishing Awareness Training
> Developed an interactive phishing awareness training web module.

- Explained phishing attack types (Email, Spear, Smishing, Vishing)
- Taught users how to identify and avoid phishing attempts
- Included real-world examples and prevention best practices
- Interactive quiz to test user awareness
- **Tools:** HTML5, CSS3, JavaScript

---

### ✅ Task 3 — Secure Coding Review
> Performed a full security audit on a vulnerable Python application.

- Identified **5 vulnerabilities** via static analysis and manual inspection
- Vulnerabilities covered: SQL Injection, Hardcoded Credentials, Path Traversal, Weak Hashing (MD5), Information Leakage
- Provided a detailed PDF report with proof-of-concept, impact, and remediation for each finding
- Delivered a fully fixed secure version of the application
- **Tools:** Python, SQLite, ReportLab, Manual Code Review
